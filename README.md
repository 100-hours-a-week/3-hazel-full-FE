# Up-Date

## Front-end 소개

- 다양한 챌린지 참여와 매일 인증을 중심으로 `사용자의 꾸준한 성장을 돕는 기록형` 프로젝트입니다.
- `Vanilla JavaScript` 라이브러리를 사용하여 구현했습니다.
- 개발은 초기 프로젝트 화면부터, 기능, 백엔드 연결까지 `직접 구현`했습니다.

### 개발 인원 및 기간

- 개발기간 :  2024-09-22 ~ 2024-12-07
- 개발 인원 : 프론트엔드/백엔드 1명 (본인)

### 사용 기술 및 tools
- HTML5
- CSS3
- Vanilla JavaScript (ES6+)
